## 짧은 설명 (80자 이내)
```
접속하지 않아도 영웅들이 자동으로 싸운다! 모으고 키우는 정통 방치형 수집 RPG.
```

## 전체 설명 (4000자 이내)
```
◆ 엘드리아 연대기 ◆
불·물·숲·빛·어둠. 다섯 속성의 영웅을 모아 끝없이 깊어지는 스테이지에 도전하세요.
접속하지 않아도 파티가 자동으로 전투하고 보상을 쌓아두는 정통 방치형 수집 RPG입니다.

▶ 손 안 대도 성장하는 방치 전투
- 자동 전투로 스테이지를 클리어하고, 오프라인에도 보상이 누적됩니다.
- 강해지면 다음 층으로 자동 전진, 벽에 막히면 그 자리에서 파밍.

▶ 모으고, 키우고, 조합하는 재미
- 노멀~신화(UR) 5등급 영웅을 소환으로 수집.
- 레벨업·돌파·각인·스킬·전용무기·룬·코스튬까지 깊은 육성.
- 장비/악세/유물/펫 — 다양한 성장 축과 세트 효과.

▶ 속성과 시너지의 전략
- 5속성 상성과 삼위일체·오색 결속 등 파티 시너지.
- 공격·방어·지원 3역할 조합으로 나만의 전투력 설계.

▶ 풍성한 콘텐츠
- 요일·속성·펫조각 던전, 스토리 캠페인, 무한의 탑, 아레나·길드.
- 일반부터 나락까지 4단계 난이도 — 고위험 고보상.

▶ 편리한 운영
- x10 / x100 / Max 일괄 강화·소환.
- 전투력 기여 요소를 한눈에 보는 분해 표.
- 자동 저장 · 이관 코드로 안전하게.

지금 첫 영웅과 함께 엘드리아의 정점에 도전하세요!
```
