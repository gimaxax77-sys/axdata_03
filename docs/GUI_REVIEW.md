# GUI 냉정 리뷰 — 2026-07-24

> 기준: 방치형 수집 RPG 상위작 **세븐나이츠 키우기(넷마블) · AFK 저니 · 버섯커 키우기**.
> 목적: 응원 아닌 진단. 시스템 감사([`SYSTEM_AUDIT.md`](./SYSTEM_AUDIT.md))의 UI판.

## 0. 한 줄 총평
**정보는 다 있는데 "지금 뭘 눌러야 하나"가 약하다.** 구조는 장르 표준을 따르나,
한 화면(영웅)에 과부하가 걸려 있고, 시각 위계·타격감·첫인상이 상위작 대비 밋밋하다.

## 1. 현재 GUI 골격 (사실)
- **하단 탭 6개**: 🏰전투(방치) · 🦸영웅 · 🔮소환 · ⚔️원정 · 📅콘텐츠 · 🛒상점
- **상단 고정**: 자원바(골드·정수·소환석·다이아) + 📬우편 + 공지 배너
- **영웅 탭**: 로스터 서브탭 4(영웅/편성/성장/기록) × 상세 서브탭 4(육성/장비/스킬/꾸미기) — **한 화면에 16 뷰 상태**(RosterScreen 1266줄)
- **콘텐츠 탭**: 서브탭 4(일일/이벤트/경쟁/스토리) — 던전·이벤트+시즌·아레나+탑+길드·캠페인

## 2. 강점 (짧게)
- 6탭 하단 내비 = 장르 표준(세븐나이츠와 동형). 익숙함.
- 자원바 상시 노출, 탭 아이콘+라벨+활성 인디케이터, "받을 것" 빨간 점.
- 최근 도입한 토큰·Card 그림자로 카드 일관성·깊이 개선됨.
- 서브탭으로 최상위 탭 수를 억제(원정 흡수 등).

## 3. 냉정한 문제점 (벤치마크)

### G1. "영웅" 탭 과부하 — 내비 깊이 과다
로스터+상세+편성+진형+성장(펫/유물/엠블럼/정령)+기록을 **한 화면에 다** 넣었다.
- 유물 하나 강화하려면 **영웅 → 성장 → 유물 카드 → 상세** (탭 3~4회).
- 세븐나이츠/AFK는 영웅(육성)·장비·펫/유물을 **별도 진입점**으로 분리해 1~2탭에 도달.
→ 1266줄 단일 화면은 유지보수뿐 아니라 **유저 인지·도달성**의 병목.

### G2. "지금 할 일"이 흐릿함 — 주 루프 경쟁
- 방치 전투(전투 탭)가 주 루프인데, **무한탑**은 콘텐츠>경쟁, **원정**은 별도 탭,
  **캠페인**은 콘텐츠>스토리로 흩어져 "다음 목표"가 분산.
- 상위작은 **메인 화면 하나에 핵심 CTA**(레벨업·소환·다음 스테이지)를 크게 두고 나머지를 명확히 하위로.
→ 우리는 여러 콘텐츠가 **동등한 무게**로 경쟁해 초점이 없다. (ObjectiveBanner가 있으나 약함.)

### G3. 시각 위계·타격감 밋밋
- Card+텍스트 위주로 **평면적**. 큰 숫자·큰 캐릭터·강한 CTA 대비가 약함.
- 전투 연출은 최근 타격 충격을 넣었지만, 상위작의 **스킬 컷인·데미지 폰트·연출 강도**엔 못 미침.
- 첫인상: 우리는 **텍스트 4장 모달** 먼저. 세븐나이츠는 **스크립트 첫 전투+보상 폭우**로 "손맛" 먼저.

### G4. 세로 스크롤 과다
- 콘텐츠(497줄)·소환(325)·상점(311)·성장(309)은 **긴 세로 스크롤**.
- 상위작은 카드·캐러셀·탭으로 **한 화면 압축**, 스크롤 최소화.

### G5. 모듈 토킷과 GUI 정합(중요·현재 목표 연계)
Gim 목표(모듈 on/off 조립식)에 맞추려면 **GUI도 조립식**이어야 한다.
- 지금 게이팅으로 섹션·탭은 숨겨지나(진행 중), **레이아웃이 빈자리 없이 재구성**되는지,
  단순 프리셋에서 화면이 **깔끔히 축소**되는지 설계 필요.
- 즉 "모듈 끄면 그 UI 자리가 자연스럽게 사라지고 남은 게 채워지는" **적응형 레이아웃**이 다음 과제.

## 4. 요소별 (냉정)
| 영역 | 상태 | 벤치마크 대비 |
|---|---|---|
| 하단 내비 | ✅ 표준 | 동급 |
| 자원바/뱃지 | ✅ | 동급(아이콘 퀄은 아트 의존) |
| 영웅/성장 화면 | ⚠️ 과부하(G1) | 하회 — 분리 필요 |
| 메인 루프 초점 | ⚠️ 분산(G2) | 하회 — CTA 강화 |
| 시각 위계·연출 | ⚠️ 밋밋(G3) | 하회 |
| 스크롤/압축 | ⚠️ 김(G4) | 하회 |
| 일관성(토큰) | 🟡 개선중 | 근접 |
| 조립식 적응 레이아웃 | ❌ 미설계(G5) | 신규 과제 |

## 5. 우선순위 제언
1. 🔴 **G2 메인 루프 초점화** — 전투(메인) 화면에 "다음 목표" CTA를 크게. ObjectiveBanner 강화·상단 고정.
2. 🔴 **G1 영웅 탭 분해(UI)** — 성장(펫/유물/엠블럼/정령)을 영웅에서 떼어 별도 진입점 또는 상위 서브탭으로. (코드 분해는 이미 논의됨 — UI 재배치와 함께.)
3. 🟡 **G5 적응형 레이아웃** — 모듈 off 시 빈자리 없이 재구성되는 규칙. 토킷 목표의 GUI 완성.
4. 🟡 **G3 연출·위계** — 큰 숫자·CTA 대비, 전투 스킬 컷인(아트 합류 시).
5. 🟢 **G4 스크롤 압축** — 긴 화면을 카드/캐러셀로.
6. 🟢 **첫 전투 온보딩** — 텍스트 모달 대신 스크립트 첫 전투+보상.

> 요약: 골격은 장르 표준. 약점은 **초점(무엇을 먼저)·영웅탭 과부하·연출 강도**, 그리고
> 토킷 목표를 위한 **조립식 적응형 레이아웃**이 아직 없다는 점.
