⚠ 고를 때 주의 (수정작업 감안)

  • 저해상 모듈러 제외 — Mana Seed·xiaosuzaoshui·LPC는 캐릭터가 작아(16~32px) 크게 띄우면 뭉개짐. 확대해도 디테일 안 늘어남.
  • "256px ≠ 항상 고디테일" — 네이티브 대형(PixelDuck)은 진짜 고디테일, 확대본(Beowulf 256)은 64의 정수배(같은 디테일·큰 크기). 실제 디테일을 보고 고르기.
  • LPC 라이선스 함정 — GPL/CC-BY-SA + 출처표기 + DRM/앱스토어 충돌 → 상업 모바일 부적합. 편해도 회피.
  • 안전 등급 — CC0 > 무료·상업(표기불요) > 상업(표기). CC-BY-SA·GPL은 배제.
