풀 비주얼 픽셀 게임용 카테고리별 추천. **다운로드는 Gim이 itch.io 로그인 후**,
받은 파일은 제가 우리 규격으로 렌더/통합합니다. **라이선스는 각 페이지에서 최종 확인** 필수.

> ⚠️ 3대 주의
> 1. **스타일 통일** — 여러 작가 팩을 섞으면 그림체가 튑니다. **캐릭터 팩 1개를 기준**으로
>    해상도·톤을 맞춰 UI·몬스터·아이콘을 고르세요.
> 2. **시점** — 우리 전투는 좌우 대치(정면/사이드). 탑다운 걷기 스프라이트는 정면 대기컷만
>    쓰거나 부적합할 수 있음.
> 3. **로스터 규모** — 영웅 28종 + 등급 + 적 다수. 아래 "대형 팩"으로 한 방에 확보 권장.

## 8. 편집 용이성 (수정작업 관점) — 신중 분석

우리 파이프라인은 **색변형·틴트·아웃라인·조립**을 코드로 자동화합니다. 그래서 팩을 고를 때
"얼마나 편집하기 쉬운가"가 핵심입니다. 아래 4축으로 평가:

- **소스파일**: Aseprite/PSD 레이어 제공 → 파츠 편집·의상 변형 쉬움
- **모듈러**: base + 파츠(옷/헤어/무기) 구조 → 1개로 다수 변형 생성
- **해상도**: 높을수록 디테일 추가/재작화 여지 (저해상은 확대해도 디테일 X)
- **수정 라이선스**: 개조 허용 + 상업 클로즈드(모바일)에 안전한가

| 팩 | 소스파일 | 모듈러 | 해상도 | 수정 라이선스 | 편집 난이도 |
|---|---|---|---|---|---|
| **Mana Seed Character Base** (Seliel) | ✅ Aseprite 레이어 | ✅ 의상/헤어 | 중(사이드뷰) | 무료·상업 | ★쉬움 |
| **Free Pixel Art** (xiaosuzaoshui) | ✅ 레이어드+팔레트 | ✅ 무기/망토/액세 | 중 | royalty-free | ★쉬움 |
| **Character Templates** (ErisEsra) | ✅ 템플릿 | ◐ 빈 캐릭에 착장 | 중 | 상업(확인) | 쉬움 |
| **Beowulf 144** | ✗ 완성 스프라이트만 | ✗ | **고(64~256px)** | 상업 | 색변형만 쉬움·구조편집 어려움 |

### ⚠️ 라이선스 함정 — 반드시 회피
- **LPC (Universal LPC Generator)** — 가장 강력한 모듈러 생성기지만 **GPL-3.0 / CC-BY-SA-3.0**:
  전 기여자 **출처표기 필수** + **ShareAlike**(2차물도 같은 라이선스 강제) + **DRM/앱스토어 충돌**
  소지. **상업 클로즈드 모바일엔 부적합.** 편해 보여도 쓰지 말 것.
- 안전 등급: **CC0 > 무료·상업(표기불요) > 상업(표기)**. `CC-BY-SA`·`GPL`은 피하기.

### 수정작업 현실 (무엇이 쉽고 무엇이 어려운가)
| 작업 | 난이도 | 조건 |
|---|---|---|
| 속성·등급 **색변형/틴트** | ★자동 | 아무 팩 OK (팔레트 매핑) |
| **아웃라인·그림자·글로우** | ★자동 | 코드 셰이딩 |
| **파츠 조립·의상 변형** | 중 | **소스파일+모듈러** 필요(Mana Seed·xiaosuzaoshui) |
| **디테일 추가/재작화** | 높음 | **고해상 원본** 필요(Beowulf). 저해상 확대는 불가 |
| **시점 변환**(탑다운→정면) | 매우 높음 | 사실상 재작화 → 처음부터 정면/사이드 팩 고르기 |

### ⚠️ 해상도 정정 — 모듈러 무료팩은 대부분 저해상
Mana Seed·xiaosuzaoshui·LPC 계열은 **탑다운 RPG용이라 캐릭터가 작습니다(≈16~32px)**.
편집은 쉬워도 **우리처럼 캐릭터를 크게 띄우는 화면에선 뭉개집니다.** 확대해도 디테일이
안 늘어남 → **"풀 비주얼"이면 모듈러보다 해상도가 우선.**

우리 게임은 **초상(로스터/소환 카드) + 배틀러(전투 무대 풀바디)** 두 표시가 필요:
