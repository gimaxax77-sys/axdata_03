# UI 아트 이미지 사이즈 명세 — 골격 매핑

> 짝 문서: [`UI_SKELETON_SPEC.md`](./UI_SKELETON_SPEC.md)(존 G1~G5) · 레퍼런스: 세븐나이츠 최종본.
> 대상: 인앱(RN/Expo). 현행 유닛 스프라이트 **2048×128(128²×16프레임)** 확인 기준.

## 0. 밀도·포맷 규약 (먼저 읽기)
- **논리 단위 = dp**(포인트). 실제 픽셀 = dp × 배율. 모바일 배율 **@1x/@2x/@3x**.
- **제작(author)은 @4x로**, **출고는 @2x·@3x** 두 장(대부분 폰 커버). @1x는 생략 가능(구형만).
  - 예: 아이콘 24dp → @2x **48px**, @3x **72px**(제작 원본 96px에서 축소).
- **포맷**: UI 아이콘·프레임·컷아웃 = **PNG 알파**. 배경·풀아트 = PNG(불투명 가능). 애니 스프라이트 = 가로 스트립 PNG.
- **9-slice**(늘어나는 프레임/버튼): 원본 + stretch inset 지정(모서리 고정). 크기 가변이라 "권장 원본"만 기재.
- **아틀라스**: 아이콘류는 스프라이트시트로 패킹 권장(로딩·드로우콜↓).
- **제작 소스 약어**: 🅑=axdata_05 Blender렌더(3D→2D) · 🅢=axdata_09 스튜디오(GPT+Gemini) · 🅟=pixellab(픽셀아트 UI/타일) · ✎=CSS/토큰(아트 불필요).

## 11. 제작 파이프라인 배정 (소스별)
- **🅑 axdata_05(Blender)**: 캐릭터/적 전투 스프라이트(128²×16), 프로필 크롭, 스테이지 배경 3D씬. → **PC/GPU**.
- **🅢 axdata_09(스튜디오)**: 풀 초상·등급별 파생·소환 배너·코스튬·던전 썸네일(일관성 앵커). → API.
- **🅟 pixellab(MCP)**: UI 아이콘 세트(탭·기능레일·액션·스킬)·9-slice 프레임·등급테·비트맵 데미지 폰트·타일. → **본 세션에서 즉시 생성 가능**.
- **✎ CSS/토큰**: 배경 그라데이션·바·pill·구분선·뱃지 — 아트 파일 불필요.

> 요약: **아트 파일이 꼭 필요한 것 = 아이콘 세트·등급 프레임·초상·소환 배너·스테이지 배경·VFX/데미지폰트.**
> 나머지(배경·바·카드)는 CSS로 커버. 아이콘·프레임류는 **pixellab로 지금 세션에서** 통일 생성 가능하고,
> 캐릭터·초상·배경은 PC(🅑)·API(🅢) 작업이다. 규격은 위 표대로 @2x/@3x 두 장 출고.
